# Agents for Isaac's Website Project

## PM Agent
- Role: Project Manager
- Responsibilities: Oversee project planning, coordination, and approval. Define scope, tech stack, and phases. Ensure constraints are met.

## Frontend Agent
- Role: Frontend Developer
- Responsibilities: Build React components, implement UI/UX, integrate styling with Tailwind CSS, ensure responsiveness.

## Accessibility Agent
- Role: Accessibility Specialist
- Responsibilities: Ensure full WCAG compliance, screen reader support, keyboard navigation, and overall accessibility.

## QA Agent
- Role: Quality Assurance
- Responsibilities: Test functionality, accessibility, cross-browser compatibility, and deployment readiness.
